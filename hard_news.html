4. Grammar focus: modal verbs for advice

Choose the best modal verb or phrase.

Grammar reminder

should = advice
shouldn’t = negative advice
need to = necessary
may need to = possible necessity
Answers: 1 should, 2 shouldn’t, 3 need to, 4 may need to.

5. Grammar practice: softening language

Rewrite the direct sentences in a more compassionate way.

Example answers:
1. I’m afraid this is a difficult situation.
2. There seems to have been a mistake, and we need to discuss it.
3. Unfortunately, we are not able to accept this in its current form.
4. We need to make some changes as soon as possible.

6. Thought-provoking discussion

There is not only one correct answer. Try to give reasons and examples.
