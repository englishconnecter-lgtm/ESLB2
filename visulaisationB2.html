4. Grammar focus: can, could, might

Choose the best modal verb. Think about ability, possibility and cautious claims.

Grammar reminder

can = ability or general possibility
could = possible result
might = less certain possibility
Use modals + base verb
Answers: 1 can, 2 could, 3 might, 4 can.

5. Grammar practice: second conditional

Complete the sentences with your own ideas. Use this structure: If + past simple, would + verb.

Example answers:
1. If I visualised my goal every morning, I would feel more motivated.
2. If I had to give an important presentation, I would imagine myself speaking calmly.
3. If a person imagined failure too often, they would probably feel more anxious.
4. If schools taught mental preparation, students would feel more confident before exams.
